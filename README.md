# 🛹 Skate Street

Skate Street é um jogo 2D desenvolvido em **Python** utilizando a biblioteca **Pygame**. O objetivo é controlar um skatista, desviar de obstáculos e sobreviver o maior tempo possível para alcançar a maior pontuação.

Este projeto foi desenvolvido com foco na programação orientada a objetos e manipulação de eventos em Python.

---

## 🎮 Funcionalidades

- Menu inicial interativo
- Sistema de pontuação
- Dois níveis de dificuldade
- Obstáculos dinâmicos
- Sistema de Game Over
- Ranking de pontuação
- Música de fundo
- Efeitos sonoros
- Animações do personagem
- Progressão entre fases

---

## 🛠 Tecnologias Utilizadas

- Python 3
- Pygame
- Programação Orientada a Objetos (POO)

## 🎯 Objetivo

O jogador deve controlar o skatista, evitando obstáculos durante o percurso para acumular a maior pontuação possível. Conforme o progresso, o jogo apresenta novos desafios e aumento da dificuldade.
